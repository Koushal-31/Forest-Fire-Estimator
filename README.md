# 🔥 Forest Fire Prediction using Machine Learning

A machine learning project that predicts the likelihood of forest fires using environmental and weather-related parameters.

## 📌 Project Overview

This project analyzes historical forest fire data and builds a predictive machine learning model.

## 📂 Files

- `forestfireestimator.ipynb` – Complete Jupyter Notebook
- `forestfires.csv` – Dataset used for training and evaluation

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Workflow

1. Data Loading
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction

## 🎯 Objective

To predict the occurrence and severity of forest fires using machine learning techniques.
